README - Transferência de Aprendizado e Fine-Tuning para Classificação de Imagens de Doenças de Pele
Este projeto explora o uso de redes convolucionais (CNNs) para classificação de imagens relacionadas a doenças de pele, utilizando diferentes estratégias de aprendizado de máquina, como treinamento do zero, transferência de aprendizado e fine-tuning. O objetivo é avaliar como essas abordagens impactam o desempenho do modelo, dadas as características desafiadoras do dataset, como a alta variabilidade das imagens.
